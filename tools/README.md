# FigTabMiner Tools

## Available Tools

### 1. visualize_results.py
Generate HTML report of extraction results.

```bash
python tools/visualize_results.py
```

### 2. evaluate_accuracy.py
Evaluate system accuracy using annotated dataset.

```bash
python tools/evaluate_accuracy.py
```

### 3. diagnose_accuracy.py
Diagnose detection issues on specific images.

```bash
python tools/diagnose_accuracy.py <image_path>
```

### 4. annotation_tool.py
Create and edit annotations for evaluation.

```bash
python tools/annotation_tool.py --doc-id <doc_id>
```
