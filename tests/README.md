# FigTabMiner Tests

## Running Tests

```bash
# Run all tests
bash tests/run_tests.sh

# Or use pytest directly
pytest tests/ -v

# Run specific test file
pytest tests/test_models.py -v
```

## Test Files

- `test_models.py` - Core data models
- `test_detection_fusion.py` - Detection fusion logic
- `test_enhanced_ai_analyzer.py` - AI analysis
- `test_enhanced_bbox_merger.py` - Bounding box merging
- `test_enhanced_chart_classifier.py` - Chart classification
- `test_enhanced_quality_assessor.py` - Quality assessment
